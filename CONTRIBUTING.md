# Contribuindo para Bíblia Multi-Idiomas (Electron)

Obrigado por querer contribuir! Este documento descreve como propor melhorias e enviar alterações com segurança.

---

## ✅ Antes de começar

- Mantenha as alterações pequenas e objetivas.
- Abra uma issue (ou descreva no PR) explicando:
  - O problema/necessidade
  - A solução proposta
  - Impacto esperado

---

## 🧰 Requisitos (dev)

- Node.js LTS
- npm
- Dependências do Electron instaladas no sistema (Linux)

---

## 🚀 Rodar localmente

```bash
npm install
npm run dev
